# BAC Backend

## Local Setup
1. Install dependencies: `npm install`
2. Start server: `npm start`

## Endpoints
- GET /health
- GET /api/usage/summary
- GET /api/usage/data
- GET /api/usage/insights
- GET /api/sensor-data
- POST /api/command

## Deploy to Render
- Build Command: `npm install`
- Start Command: `npm start`
